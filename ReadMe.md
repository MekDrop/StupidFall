Game goal
-----------
  Try to stay alive as long as possible.

Controls
-----------

  Right - move right<br />
  ESC - exit

-----------
Kiek galima ilgiau išsilaikyti ir surinkti kiek galima daugiau taškų.

-----------
  Kairė - juda kairėn<br />
  Dešinė - juda dešinėn<br />
  ESC - išeiti (lango uždarymas neveikia)
